# Random Access Labs

> Where "What if?" becomes "What's next?"

Random Access Labs is an independent software studio based in Australia. We focus on the intersection of minimalist design and high-precision logic, specializing in "Zero Draft" development—the process of moving from a core concept to a production-ready application with zero bloat and maximum performance.

_Directly accessing innovation at the speed of O(1) ⚡_

### Current Production

**SYLO**
*Type: Mobile Game*
A geometric noir experience focused on high-precision rhythmic timing and physics-based movement. Built on a custom engine to ensure responsive, kinetic feedback.
* **Platform:** iOS and Android
* **Status:** Available on [![App Store](https://img.shields.io/badge/App_Store-000000?logo=apple&logoColor=white)](#) and [![Google Play](https://img.shields.io/badge/Google_Play-414141?logo=google-play&logoColor=white)](#)
* **Stack:** React Native, Expo, Zustand

#

**QRido**
*Type: Web Application (Utility)*
A privacy-focused, minimalist utility for instant QR code generation. Designed for speed with no tracking, no advertisements, and no user data collection.
* **Platform:** Web
* **Status:** [Live Website](https://qrido.bharticomputer.in)
* **Stack:** Next.js, Tailwind CSS

### Technical Foundation

We utilize a modular, modern stack to maintain high iteration speed across all laboratory experiments:

* **Mobile Development:** React Native and Expo for cross-platform performance.
* **Web Development:** Next.js and Tailwind for optimized delivery.
* **State & Logic:** TypeScript for type safety and Zustand for lightweight state management.
